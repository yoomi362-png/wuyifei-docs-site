文档发布说明

以后你新增文档时，按下面这个方式做，网站会自动在对应分类里展示出来。

  1. 把新的 HTML、PDF 或文档链接准备好。站内文件可以放进 `docs/`，外部文档可以直接用完整链接。
  2. 打开网站根目录下的 `docs-data.js`。
  3. 在 `window.DOC_ITEMS` 里新增一条记录，填写 `title`、`category`、`description`、`href`、`updatedAt` 和 `tags`。
  4. 如果要新增分类，就在 `window.DOC_CATEGORIES` 里加一个新的分类对象。
  5. 提交并推送到 GitHub，网页会自动显示新的文档卡片。
