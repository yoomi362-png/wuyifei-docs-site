# wuyifei.xyz 文档站

这是一个可直接部署到 GitHub Pages / Cloudflare Pages 的静态文档站。

## 本地预览

直接打开 `index.html`，或用任意静态服务器运行。

## 部署

### GitHub Pages

1. 新建仓库并上传所有文件
2. 仓库 Settings -> Pages
3. Source 选 `Deploy from a branch`
4. Branch 选 `main`，Folder 选 `/root`
5. 自定义域名填写 `wuyifei.xyz`

### DNS

把域名 `wuyifei.xyz` 的 DNS 按 Pages 平台要求指向部署服务。
